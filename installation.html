Installation¶

Requirements¶

OpenNMT-tf requires:

  • Python (>= 2.7)
  • TensorFlow (>= 1.4)

Some features added after the initial release may require newer TensorFlow versions. Please consult the changelog to know about new features availability.

Via pip (recommended)¶

Each tagged version of the project is automatically pushed to PyPi. We recommend installing this package via pip:

pip install OpenNMT-tf

To confirm that the package is correctly installed, run:

onmt-main -h
